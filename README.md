# New World tools

## Usage

### Pak extracter

```
.\pak-extracter.exe ^
    -assets "C:\Program Files (x86)\Steam\steamapps\common\New World\assets" ^
    -output ".\extract" ^
    -filter .ext1,.ext2
```

### Datasheet converter

```
.\datasheet-converter.exe ^
    -input ".\extract" ^
    -output ".\extract\datasheets"
```
